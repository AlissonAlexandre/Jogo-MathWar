Projeto Jogo




Esse jogo foi desenvolvido para a disciplina de desenvolvimento web do Centro Universitário FEI.
A ideia principal do jogo é ser um jogo educativo usado para ajudar a realizar as operações básicas de cabeça.

O jogo foi feito em javascript, utilizando diferentes arrays com objetos criados com base na sua respectiva operação matematica (multiplicação, soma e subtração). O jogo tem três niveis de dificuldade, dois modos de jogo, controle de volume, botão de pausa e de parar o jogo, além do botão de iniciar como solicitado no projeto.
O jogo também possui seletor de modo, assim como um vídeo passando no background do canvas.




O site inicia da primeira vez com o canvas, o video de fundo, o botão de pausa, o botão de parar e o campo e botão de input escondidos.

Ao clicar no botão iniciar o canvas e o video de fundo aparecem, assim como os elementos do jogo e esconde todos os elementos que não fazem parte do jogo, como seletor de dificuldade e de modo.

Após o jogo acabar os elementos do jogo são escondidos e os outros elementos são mostrados, com exceção do tutorial de como jogar o jogo, visto que se pressupõe que o usuário já aprendeu a como jogar.
